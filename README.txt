Программа телефонный справочник
Функционал:
1. Открыть файл
2. Сохранить файл
3. Добавить контакт
4. Удалить контакт
5. Показать все контакты
6. Показать контакт
7. Изменить контакт
8. Остановить программу

Модули:
  models: 
      Класс пользователь отвечает за хранение данных ввиде пользователей(контактов) с ID
      Класс контакты, отвечает за работу с коллекцией пользователей их хранение и изменение 
  main:
      основная программа в которой собран проект
  exceptions:
      модуль отвечает за обработку кастомных исключений
  controller:
      модуль отвечате за открытие и чтение файла, а так же для открытия и записи в файл
  test_main:
      модуль для тестирования
  confest:
      модуль в котором содержаться фикстуры
